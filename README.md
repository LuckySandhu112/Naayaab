# Naayaab-e-Kala
### Where Creativity Becomes Timeless Art

Naayaab-e-Kala is a premium, responsive art studio website built using **HTML, CSS, and JavaScript**. The website blends traditional Indian artistic aesthetics with modern minimalism to create an immersive experience for visitors.

It showcases the studio's services, featured artworks, artists, workshops, testimonials, and contact information while providing a smooth, elegant, and user-friendly interface.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Naayaab-e-Kala.git
```

Open the project

```bash
cd Naayaab-e-Kala
```

Launch

Simply open **index.html** in your browser.

Or use **Live Server** in Visual Studio Code.
